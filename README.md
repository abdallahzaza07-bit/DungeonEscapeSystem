Dungeon Escape System
Description

Dungeon Escape System is a Java Swing-based dungeon game where the player must find a key, avoid a monster, collect treasure, and escape through the exit.

Features
2D Grid System
Player Movement
Monster AI Pathfinding (BFS)
Recursive Maze Generation
Difficulty Selection
Score System
Leaderboard
Controls
W A S D = Move
Arrow Keys = Move
How to Run
Open project in Eclipse.
Run DungeonEscapeSystem.java.
Select difficulty.
Escape the dungeon.
